<br />

| tag       | 설명                                           |
| --------- | ---------------------------------------------- |
| `header`  | 네비게이션이나 페이지에 대한 정보, 페이지 상단 |
| `nav`     | 네비게이션 링크들의 집합, 메뉴, 목차           |
| `aside`   | 사이드에 위치하는 공간을 의미                  |
| `section` | 본문의 여러 내용을 포함하는 공간을 의미        |
| `article` | 본문의 주내용이 들어가는 공간을 의미           |
| `footer`  | footer 페이지 하단, 페이지 관련 정보 표시      |
